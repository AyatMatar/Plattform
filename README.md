#  Plattform - Educational Platform

Plattform is a full-stack educational web application that connects students with instructors through structured courses, built using Angular for the frontend and ASP.NET Core API for the backend.

---

##  Key Features

###  User (Student)
- Browse available courses (after login)
- Subscribe to courses
- Communicate with instructors via provided links
- Access course content (videos & files)

###  Instructor
- Upload course materials (videos, files)
- Provide explanations and learning resources

###  Admin
- Create and manage courses
- Assign instructors to courses
- Manage the platform structure

---

##  Authentication & Authorization

- Implemented using **JWT (JSON Web Token)**
- Secure login system
- Role-based access (Admin / Instructor / Student)

---

##  Technologies Used

### Frontend
- Angular
- TypeScript
- HTML, CSS
- Bootstrap (template customized)

### Backend
- ASP.NET Core Web API
- C#
- Entity Framework Core

### Database
- SQL Server

---

##  System Architecture

- Frontend and Backend are built as **separate projects**
- Communication via RESTful APIs
- Token-based authentication using JWT
